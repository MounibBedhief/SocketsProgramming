# 🛠️ Environment Setup Guide

## 🪟 For Windows Users
If you are on Windows, you do not need to install a new operating system. You can use **WSL (Windows Subsystem for Linux)** with **Ubuntu**. 
Simply open PowerShell as Administrator and run `wsl --install` to get your Ubuntu environment ready.

---

## 🐧 Linux / Ubuntu Setup
Open your terminal and run the following commands one by one to install everything you need:

**1. Update your package manager:**
```bash
sudo apt update
```
2. Install the GCC compiler:
```bash
sudo apt install gcc
```

3. Install Wireshark:
```bash
sudo apt install wireshark
```
4. Install Telnet:
```bash
sudo apt install telnet
```
